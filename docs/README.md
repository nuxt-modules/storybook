## Update

The documentation is based on the [Nuxt UI Docs](https://github.com/nuxt-ui-templates/docs) template. In order to update to the latest version of the template, run the following command:

```bash
rm -rf docs
npm create nuxt@latest -- -t ui/docs docs
```
