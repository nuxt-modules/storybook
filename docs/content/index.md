[Storybook](https://storybook.js.org/) integration with [NuxtJS](https://nuxtjs.org).

## Features

::list
- Zero configuration
- Nuxt webpack configuration
- Nuxt plugins support
- Story discovery from nuxt modules
- Nuxt components support
- Storybook Generate
- Hot reload support
::
